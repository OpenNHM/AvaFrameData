# avaArzl 

## Event 201901

### General description 
- Type of avalanche: dense flow with major powder cloud, dry snow
- Entrainment: Significant for avalanche formation
- Runout: The maximum avalanche runout reached the edge of the base of the catchment dam at approx. 945m a.s.l. 
- Date: .01.2009 

### Available data
- release area shp file (releaseWLV.shp)
    - release height 1,76m
- entrainment area shp file (entrainmentWLV.shp)
    - entrainment height 0,3m
- forest shp file (forest.shp)
    - used features from Waldkategorienlayer for resistance areas classified as:
        - Schutzwald ausser Ertrag
        - Schutzwald im Ertrag
        - Wirtschaftswald
        - Wirtschaftswald, mittlere Schutzfunktion
- digital terrain model (DEM5mArzl.asc)
- digital terrain model projection (DEM5mArzl.prj)
